WarpWallet

What is WarpWallet?

WarpWallet is a deterministic bitcoin wallet generator. You pick a unique passphrase which is big, but which you can remember. You never have to save or store your bitcoin wallet anywhere -- online or offline. Just remember your passphrase. Then send money to the public address. When you want to extract the BTC (and import to an online wallet), just re-generate your private key.

This is not an original idea. bitaddress.org's brainwallet is our inspiration.

WarpWallet adds two improvements: (1) you can salt your passphrase with your email address. This is trivial for you to remember, so why not? It is optional of course. And (2) WarpWallet uses scrypt to make address generation both memory and time-intensive.

You can save this self-contained HTML file and access it wherever you like, even on an air-gapped computer. Heck, email it to 10 friends if you're afraid it will go away. All the JS, CSS, and images are embedded.

If you're a programmer and want to reproduce WarpWallet's output, it's simply: 

  # max to write this part, filler
  bleah(pbkdf2(scrypt(passphrase, salt)))

WarpWallet Challenge

We would like to get an idea of the bitcoin community's ability to crack WarpWallet, so we are making 50 bitcoins available should you crack it.

The first couple levels should go quickly. We hope it is impossible to crack a 9 character passphrase by brute force. If we're wrong, and that money is stolen, oh shit, basically. Scrypt is fucked.

We generated 5 addresses using unsalted passphrases. All the passphrases are alphanumerics only. For example: 'hY9f' would be a legit 4 character passphrase.
