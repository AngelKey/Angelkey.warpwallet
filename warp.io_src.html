What is WarpWallet?

WarpWallet is a deterministic bitcoin address generator. You never have to save or store your private key anywhere. Just pick a really good, unique password.

This HTML is self-contained for portability. Access it wherever you like - including an air-gapped machine. And, heck, email it to 10 friends if you're afraid it will go away. All the JS, CSS, and images are embedded. It's also hosted on github.

This is not an original idea. bitaddress.org's brainwallet is our inspiration.

WarpWallet adds two improvements: (1) WarpWallet uses scrypt to make address generation both memory and time-intensive. And (2) you can salt your passphrase with your email address. This is trivial for you to remember, so why not? It is optional of course.

For safety, we've implemented WarpWallet outside of JavaScript and confirmed it generates the same output. (It's a step in our test suite.) If you're a programmer and want to implement WarpWallet yourself...
